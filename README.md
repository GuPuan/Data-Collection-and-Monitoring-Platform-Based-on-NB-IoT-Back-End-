# IoT Data Collection & Monitoring Platform (Back-End)

## Overview
This is the back-end service of the IoT data collection and monitoring platform, developed with Spring Boot. It handles NB-IoT sensor data, stores it in MySQL, and offers APIs for data control and management.

## Features
- **API Integration**: Handles data from NB-IoT sensors via APIs
- **Data Storage**: MySQL database for managing sensor data
- **CRUD Operations**: Control data collection, warnings, and more

## Installation
1. Clone this repository.
2. Set up the MySQL database.
3. Configure application properties (e.g., database credentials).
4. Build and run the application:
   ```bash
   mvn spring-boot:run
